# Product Overview

## Product Summary
yanosea個人ポートフォリオサイト - 技術ブログ、プロフィール、プロジェクト紹介を統合したパーソナルウェブサイト（https://yanosea.org）

## Core Features

### Primary Features
- **技術ブログ**: MDXベースのブログシステム（タグ、ページネーション、RSS対応）
- **Spotify連携**: リアルタイム音楽ステータス表示（現在再生中/最後に再生した楽曲）
- **プロフィール**: About ページでの自己紹介と興味・関心の表示
- **リンク集**: ソーシャルメディア・プロジェクトへのリンク管理
- **多言語対応**: 日本語・英語でのコンテンツ提供

### Technical Features  
- **レスポンシブデザイン**: モバイルファースト設計
- **ダークモード**: ライト/ダークテーマ切り替え
- **SEO最適化**: サイトマップ、RSS、OpenGraph対応
- **パフォーマンス**: 静的サイト生成による高速ロード
- **アクセシビリティ**: 日本語フォント最適化（DotGothic16、Zen Kaku Gothic New）

### Content Management
- **Content Collections**: Astroのコンテンツ管理システム活用
- **MDX記事**: Markdown + React コンポーネント組み込み
- **画像最適化**: WebP形式、レスポンシブ画像対応
- **タグシステム**: ブログ記事の分類・フィルタリング

## Target Use Cases

### Primary Use Case: **技術者向けポートフォリオ**
- ソフトウェアエンジニアとしてのスキル・経験アピール
- 技術記事での知見共有とSEO効果
- プロジェクト成果物の展示
- 採用・転職活動でのプレゼンス向上

### Secondary Use Cases:
- **個人ブランディング**: 技術コミュニティでの認知度向上
- **アウトプット習慣**: 学習内容の記録と整理
- **音楽趣向共有**: Spotify連携による個人的な趣向表現
- **技術実験場**: 新技術・ライブラリの試行プラットフォーム

## Key Value Proposition

### For Visitors:
- **高品質コンテンツ**: 実践的な技術記事・ノウハウ
- **パーソナルタッチ**: Spotify連携による人間味のある表現
- **優れたUX**: 快適な読書体験（フォント、テーマ、レスポンシブ）
- **継続性**: 定期的な更新・技術記事投稿

### For Owner (yanosea):
- **技術力向上**: ブログ執筆を通じた学習促進
- **プレゼンス構築**: Google検索・SNSでの発見性向上
- **技術実験**: 新しい技術スタックの実践的習得
- **キャリア支援**: 技術的アウトプットによる転職・採用優位性

## Business Context

### Purpose: **Personal Branding & Technical Demonstration**
技術者としてのスキルセット、思考プロセス、継続的学習能力を対外的にアピール

### Success Metrics:
- ブログ記事のPV数・滞在時間
- 検索エンジンでの技術記事ランキング
- SNSでのシェア・エンゲージメント
- ポートフォリオ経由での問い合わせ・オファー

### Long-term Vision:
技術コミュニティで認知される技術者として、継続的なアウトプットと知見共有を通じたパーソナルブランドの確立
